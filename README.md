# Zeno Api Gateway

Api gateway implemented lumen + swoole

## Requirements
>
> * PHP >= 7.4
>
> * Redis / Memcache
>
> * PHP Swoole Extension
>
> * PHP Redis Extension
>

## Features
- Authentication
- Aggregate Actions
- Resource Limiter
- Response Cache
- Manageable Auth, Route, Actions and Services
